# MyRs
MyTikTok为暑期大作业工程

## 大作业内容

  完成了一款视频播放软件，支持从网络获取视频数据和相关信息，自由播放视频，自由上传本机或者实时拍摄视频。

## 组员分工

### 卿云鹏：

  #### UI设计
  
  主页UI设计，视频播放UI设计，部分视频上传界面UI设计
  
  #### 功能需求
  
  ##### 网络数据读取和显示功能：
  
  1.使用RecyclerView显示视频列表
  
  2.使用Glide加载封面图 
  
  3.根据网络数据加载上传者姓名。
  
  ##### 视频播放功能：
  
  1.从视频信息流点击某个视频封面进入播放页面
  
  2.根据视频信息的url播放视频
  
  3.单击视频窗口暂停/继续
  
  4.双击屏幕点赞并保留点赞结果。
  
  #### 视频上传功能：
  
  从首页点击”拍摄”按钮进入拍摄页面。
  
  #### 其他
  
  所有的代码整合工作和调试工作，部分PPT设计和展示准备。
  
###  區俊豪

  #### 视频拍摄录制

2.视频录制结束之后自动进入上传页面。封面图自动生成，自动上传
